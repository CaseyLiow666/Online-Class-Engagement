# Online Class Engagement System

> Final Year Project | Diploma in Information Technology

## 📖 Overview

The Online Class Engagement System is a web-based application developed as my Final Year Project for the Diploma in Information Technology.

The system aims to improve student participation and engagement during online learning by providing lecturers with interactive classroom management tools such as attendance tracking, quizzes, polls, and engagement reports. It also offers separate modules for administrators, lecturers, and students to support efficient online class management.

---

## ✨ Features

### 👨‍🎓 Student
- Register and log in
- Join online classes
- Participate in quizzes and polls
- View personal engagement records

### 👨‍🏫 Lecturer
- Create and manage classes
- Record student attendance
- Create quizzes and polls
- Monitor student participation
- View engagement reports

### 👨‍💼 Administrator
- Manage lecturer and student accounts
- Monitor system usage
- Generate reports
